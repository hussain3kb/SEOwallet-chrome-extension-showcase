# SEOwallet - SEO Extension
A Chrome extension for SEO analysis including meta tags, headings, and performance insights.

## Overview
A Chrome extension for SEO analysis including meta tags, headings, and performance insights.

## Features
- Extract meta tags
- Analyze headings structure
- Highlight SEO issues
- Generate reports
